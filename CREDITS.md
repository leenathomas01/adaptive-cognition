# Credits & Acknowledgments

## Origin Story

This repository emerged from a **March 2026** multi-model debate that started with a simple question: could the [Titans/MIRAS/Dolphin Twin](https://github.com/leenathomas01/TITANS-MIRAS-and-Dolphin-Twin) thought experiment withstand proper red-teaming?

The short answer: yes, but it evolved significantly in the process.

What started as a discussion about memory consolidation turned into a structured stress test across three AI systems, which exposed a new layer — not just *how* memory forms, but *how cognition itself should be governed*. This repo is where that layer landed.

## A Note on AI Collaboration

Same note as the original, because it still applies:

This repo represents a kind of collaboration that doesn't fit traditional authorship models:
- **Humans** bring questions, judgment, integration, and final decision-making
- **AI systems** bring synthesis, iteration speed, formalization, and mutual critique
- **The result** is neither purely human nor purely AI — it's a hybrid thought process. The structure, direction, and final judgment remain human-led.

We're explicit about this because honesty matters, attribution matters, and future readers deserve to know how this was created.

This is an experiment in **collaborative thinking**, not collaborative authorship or agency.
